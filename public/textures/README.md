# Textures Directory

Place texture files here for Three.js materials (`.jpg`, `.png`, `.hdr`, `.exr`).

## Recommended Textures for Cosmic Theme

- Space/nebula environment maps (`.hdr` or equirectangular `.jpg`)
- Star field textures
- Planet surface textures
- Particle sprites (small `.png` with alpha)
- Normal maps for surface detail

## Optimization Tips

- Use `.webp` for 2D textures when possible (smaller file size)
- Use power-of-two dimensions (512×512, 1024×1024, 2048×2048)
- Compress HDR environment maps with [HDR Compress](https://github.com/nicoptere/hdr-compress)
- Keep individual textures under 1MB where possible
